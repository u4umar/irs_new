4 #Mixins Mixins

4.1 #Mixins.Clearfix Clearfix

Adds an ::after pseudoelement to clear floated child elements.

This may already be accounted for by Bourbon's framework but we wanted to show an example of how abstractions are used.

4.2 #Mixins.ExposedFilter Exposed Filter

Shared styling for an exposed-filter element.

4.3 #Mixins.FontSize Font-Size

  • $pxValue
    A pixel value (e.g. 1px)

Used to set the component-level font-size using a rem value and providing an em fallback using Bourbon's conversion functions. Those functions, and by extension this mixin, depend on the $em-base Bourbon variable to calculate the component font-size based on the root font-size.

Thus, this mixin cannot be used for the root element or components whose parent font-size is not the root element (e.g. a view within a view).

4.4 #Mixins.HideBlockBorder Hide Block Border

Adds a media query to the element to remove the border-top on mobile devices. This is only necessary for blocks, which have a set border-top by default in this theme for mobile devices.

4.5 #Mixins.HideBlockTitle Hide Block Title

Hides the element, supoosedly the block title element, text using Bourbon's hide-text mixin.

4.6 #Mixins.SubpageOverviewList Subpage Overview List

Styles the list of pages as flippable cards. This is also used for office location lists. The back face uses another view, but to style it similarly specific rules need to be used to override.

4.7 #Mixins.UnstyledList Unstyled List

Removes spacing and list-style of element and children li elements.
